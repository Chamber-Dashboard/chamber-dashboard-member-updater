Let your members update their own businesses on your chamber of commerce website or other membership organizations.

== Description ==
Requires Chamber Dashboard Member Manager


= Feature List:  =
* Enables user registration on your site and the users are connected to their business.
* Once the user is approved by the site admin, they can login from the front end and edit their business.

You can learn more at [Chamber Dashboard](https://chamberdashboard.com)

== Installation ==
= Using The WordPress Dashboard =

1. Download chamber-dashboard-member-updater.zip
2. Navigate to the \'Add New\' in the plugins dashboard
3. Click \'Upload Plugin\'
4. Find the file chamber-dashboard-member-updater.zip
5. Click \'Install Now\'
6. Activate the plugin on the Plugin dashboard

= Using FTP =

1. Download chamber-dashboard-member-updater.zip
2. Extract the `chamber-dashboard-member-updater` directory to your computer
3. Upload the `chamber-dashboard-member-updater` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
